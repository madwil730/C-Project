# 7-17 구조체
* 값 형식 이며 필드와 메소드를 가질 수 있다, struct 키워드를 이용해서 선언한다
* 구조체는 값 형식이기에 인스턴스는 스택에 할당되고 인스턴스가 선언된 블록이 끝나는 지점의 메모리에서 사라진다
* readOnly 키워드를 사용하여 값을 읽을 수 만 있는 struct를 만들 수 있다
```
구조체  예)

struct MyStruct
{
   //필드
   // 메소드
}

```
