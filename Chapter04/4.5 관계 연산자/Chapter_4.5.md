# 4-5 관계 연산자
두 피연산자 사이의 관계를 평가하는 연산자  

![캡처](https://github.com/madwil730/C-Project/assets/65881884/73691403-f08d-4930-a8ab-e25c228ebbfc)

```
사용 예
bool result;
result = 3 > 4; // false
result = 3 >= 4; // false
result = 3 < 4; // true
result = 3 <= 4; // true
result = 3 == 4; // false
result = 3 != 4; // true

```
