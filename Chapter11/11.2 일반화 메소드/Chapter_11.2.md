# 10.1 All for one, one for all
* 배열 : 동일한 타입의 변수를 모아놓은 데이터 집합, 배열을 통해 여러개의 데이터를 담은 변수를 만들 수 있다
* 배열의 인덱스는 1이 아닌 0 부터 시작한다
* ^키워드를 통해 배열의 마지막 인덱스에 접근 할 수 있다
```
예시)
int[] scores = new int[5].
scores[0] = 80;
scores[1] = 80;
scores[2] = 80;
scores[3] = 80;
scores[4] = 80;

scores[^1] = 10; // scores[4] = 10; 와 동일 

```
